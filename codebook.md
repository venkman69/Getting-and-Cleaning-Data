
# Code book

Note: The inertial folder values are ignored in this analysis.
* The original data has 561 fields in the test_x.txt and test_y.txt.
* Out of which only the fields matching 'std' and 'mean' are picked.
* The data in the UCI does not contain field names and in the tidy data set the data headers will be set appropriately to indicate the name.
* The clean names employed are a minor change from the original to improve readability. And the only modification is the removal of the '(' and ')' characters.
* The fields are extracted and subject and activity names are merged into to produce a single data frame with each row containing a full set of required information
* From the above data set a second, independent tidy data set with the average of each variable for each activity and each subject is created and sorted.
This data set is written to a file "tidy.txt"
